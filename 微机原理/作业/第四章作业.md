## 📝4.8

(1). AX低四位置零

```asm
and ax, fff0H
```

(2). BX低四位置一

```asm
or ax, 000fH
```

(3).AL寄存器低四位不变，高四位取反

```asm
xor al, f0H
```

(4).测试BX D1和D2位，当这两位同时为0时，将AL置0FFH，否则AL清零

(5).测试BX中 D1和D2位，当这两位有一位为0时将AL置FFH，否则AL清零

(6).将AL中保存的字母的ASCII码变换成相应大写字母的ASCII码

```asm
and al, 5fh
```

(7).将AL中保存的字母的ASCII码变换成相应小写字母的ASCII码

```asm
or al, 20h
```
